# Progressive Engagement Pipeline

Clowder is organized around a natural assessment progression. Each phase feeds evidence and context into the next phase.

```text
OSINT Recon
  -> Deep Recon
  -> Security Posture Assessment
  -> Vulnerability Assessment
  -> Red-Team / Simulated Attacker
  -> Destructive Testing Pass
```

## Phase rules

Each phase has allowed actions, blocked actions, approval rules, expected outputs, and escalation criteria.

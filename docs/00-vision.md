# Clowder Vision

Clowder is a local-first AI-assisted security assessment workbench for authorized testing by knowledgeable operators.

The project exists to provide an open, auditable, privacy-preserving alternative to opaque hosted security assessment platforms. Client data, target details, findings, evidence, and reports should remain under the operator's control.

## Core principles

- Local first
- Privacy by design
- Open and auditable
- Scope constrained
- Evidence backed
- Human supervised

## Product thesis

Clowder coordinates security assessment tools through a human-supervised AI conductor. It operates inside an explicit engagement scope, records meaningful actions, preserves evidence, and produces findings that can be reviewed, repeated, and challenged.

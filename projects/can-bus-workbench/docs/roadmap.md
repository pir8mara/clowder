# Roadmap

## 0.1: Passive starter kit

- JSONL frame format.
- candump import.
- SocketCAN capture.
- DBC decode.
- Markdown reports.
- Local LLM prompt packs.

## 0.2: Better reverse-engineering support

- Experiment metadata files.
- Compare two captures.
- Find IDs that changed between baseline and action captures.
- Candidate counter/checksum detection.
- SQLite storage for larger sessions.

## 0.3: Local agent support

- LM Studio/Ollama prompt runner.
- Chunked capture summaries.
- Hypothesis tracker.
- DBC draft generator with human approval.

## 0.4: Bench-only active testing

- Explicit safety gate.
- Dry-run mode.
- Allowlisted IDs only.
- Rate limiting.
- Reproducible replay files.
- Big warnings, because physical systems do not forgive vibes.
